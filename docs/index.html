Introduction

What is BulmaBoilerplate

BulmaBoilerplate is a profesionnal Bulma plate with Sass and JavaScript all runned by Gulp and Webpack. It provides a full project structure to help you with Bulma customization and JavaScript development.

Demo site and documentation are using Hugo, the powerful static site generator.

Features

  • Use markdown to generate a powered by Hugo documentation
  • Easily create a demo site with the power of Hugo
